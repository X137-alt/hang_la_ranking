# 控制自动化考研院校难度精细排名

GitHub Pages 静态网页版本，参考学校 Logo 拖拽排名工具的交互形式，主题改为控制/自动化考研院校难度精细排名。

## 目录结构

```text
school_logo_ranking/
├─ index.html
├─ style.css
├─ script.js
├─ README.md
├─ .nojekyll
└─ assets/
   └─ logos/
      ├─ 清华大学.png
      ├─ 浙江大学.png
      └─ ...
```

## 本版调整

- 院校卡片只显示 Logo 和院校名称。
- 移除了页面内单独的“展示缩放”按钮，整体大小跟随浏览器 Ctrl+滚轮 / 触控板缩放。
- 六栏排名面板使用等分响应式布局，浏览器缩放或窗口宽度变化时会自动重新计算卡片位置。
- 保留拖拽、自动保存、重置排名、自动整理、导出排名、直播模式、全屏展示。
- 底部说明卡片已移除。

## 使用方法

1. 把院校 Logo 放到 `assets/logos/`。
2. Logo 文件名使用院校名称，例如 `清华大学.png`。
3. 将本目录里的文件上传到 GitHub 仓库根目录。
4. 在 GitHub Pages 中选择 `main` 分支和 `/(root)` 发布。
